### 1. Explicit Manual Handoffs

**Trigger:** Any task requiring terminal commands the agent cannot execute (DB push, Env setup).

**Trigger:** Any user-facing story completion.
**Rule:** Developers are authorized to use `browser_subagent` to visually verify flows (e.g., "Click -> Edit -> Save") rather than relying solely on code tests.

### 3. Manual Testing Checkpoints

**Trigger:** Complex hardware/permission features (Audio, Microphone).
**Rule:** Developers must provide step-by-step manual test scripts for the user to execute (e.g., verifying microphone permissions).
